# Mizuki Plugin Welcome

Mizuki Plugin Welcome 是开发组内部维护的欢迎消息插件，用于整理新成员进入群聊时的基础提示、入口引导和必要说明。

仓库名目前仍为 `Mizuki-plugin-welocme`，其中 `welocme` 是历史拼写，展示名统一写作 `Mizuki Plugin Welcome`。本仓库当前不作为公开通用插件发布。

## 项目定位

该插件面向 Mizuki Bot 的群聊入口体验，主要负责在新成员进入时提供简短、稳定、可维护的提示信息。

典型职责包括：

- 维护欢迎文本和基础说明。
- 引导用户查看帮助菜单、绑定说明或其他入口。
- 根据不同群聊场景整理欢迎模板。
- 与群聊基础插件、帮助菜单插件保持一致的入口表达。

## 当前状态

- 维护状态：开发组内部维护中。
- 使用范围：面向现有机器人环境和固定群聊场景，不保证外部环境可直接运行。
- 文档状态：暂不补公开部署说明、完整配置项或调用方式。
- 许可证状态：暂未选择开源许可证。未经明确许可，不建议再分发、商用或用于生产部署。

## 维护注意事项

- 欢迎消息应保持简短、清晰，避免把大量说明堆进首条消息。
- 涉及群规、功能说明或绑定流程时，优先放入口提示，详细内容交给文档站或帮助菜单维护。
- 与 `Mizuki-plugin-help`、`Mizuki-plugin-group`、`Mizuki-plugin-qbind` 的入口命名应保持一致。
- 如果未来重命名仓库，建议从 `welocme` 修正为 `welcome`，并同步更新文档站和相关引用。
