# Mizuki-plugin-welocme

Mizuki Bot 的欢迎消息插件。

本仓库用于维护群聊中新成员加入时的欢迎提示、基础说明和入口引导。仓库名中的 `welocme` 保留当前项目命名，实际含义为 welcome。

## 项目定位

`Mizuki-plugin-welocme` 主要面向群聊入口体验。它可以用于向新成员展示群规、功能入口、绑定说明或其他必要提示，帮助用户更快理解当前群聊中的可用能力。

可能包含的内容包括：

- 新成员欢迎文本
- 群规或说明入口
- 功能菜单跳转
- 绑定流程提示
- 不同群聊场景下的欢迎模板

具体能力请以代码实现为准。

## 维护建议

欢迎消息应尽量简短、清晰，并避免过多重复内容。涉及群规或用户协议时，建议只保留入口链接或摘要，详细内容放在文档站维护。
